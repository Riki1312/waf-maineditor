done done_all Installa strumenti

Per ospitare il sito, devi installare gli strumenti a riga di comando Firebase mediante npm (Node.js)

Installa strumenti Firebase: $ npm install -g firebase-tools

Se hai già installato gli strumenti a riga di comando Firebase, esegui di nuovo il comando di installazione per assicurarti di avere l'ultima versione

 Crea il progetto

Apri una finestra del terminale e accedi a o crea una directory per il tuo sito

Accedi a Google: $ firebase login

Avvia il progetto: $ firebase init

 Implementa il sito web

Aggiungi i file statici alla directory di implementazione (quella predefinita è public)

Implementa il sito web: $ firebase deploy
